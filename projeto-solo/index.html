Código fonte/Pré-formatação

Utilizaremos esse tipo de formatação para escrever códigos e/ou calculos que facilitem a leitura. Para isso Utilizaremos o 'code' que é uma formatação JavaScript:

nome=input("qual é o seu nome?")
idade=input("quantos anos você tem?")

Para formatações onde a quebra de linha e as indetações sejam consideradas, utilizaremos o 'pre' antes do 'code': 

            
            nome=input("qual é o seu nome?")
            idade=input("quantos anos você tem?")
